# ToolLogos 目录说明

把每个软件的真实 Logo 放到本目录，文件名按 `ProgrammingTool.rawValue` 命名。

## 命名规则

- 推荐格式：`PNG`（透明背景）
- 推荐尺寸：`512x512`（最小不低于 `128x128`）
- 命名格式：`<toolRawValue>.png`
- 例子：`codex.png`、`claudeCode.png`、`githubCopilotCLI.png`

## 可选规则

- 若暂时只有 `JPG/ICO`，也可先放同名文件，后续统一转 `PNG`。
- 若有深色/浅色双版本，可先按 `<toolRawValue>.png` 放主版本；备用图后续再扩展命名。
